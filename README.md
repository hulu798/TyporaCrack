## 安装说明

- ### Windows

  - 选择要安装的版本进行下载

  - 运行 Setup.exe 安装
  
  - 将 Crack 文件夹下的文件 `winmm.dll` 复制到安装目录下

- ### macOS

  - 下载并安装即可

- ### Linux
  - deb下载并安装即可
  - rpm下载并安装需要工具转换deb安装包(官网正版也只有deb和压缩包这两种)
